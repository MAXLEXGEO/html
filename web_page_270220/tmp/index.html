SERVICIOS

DISEÑO Y AUDITORIA DE BASES DE DATOS

Maxlex Geo ha desarrollado su propio sistema de gestión de datos geológicos y de operaciones Mineras, el cual permite manejar los datos de manera transaccional y de disponibilidad, mejorando la gestión de los datos auditable de manera permanente y por consiguiente reduciendo los costos. Además, el equipo de ADMine Tecnology realiza auditorias a bases de datos bajo las perspectiva de las mejores prácticas.

MODELAMIENTO GEOLÓGICO Y GEOTÉCNICO

Capacidad de realizar el modelamiento del yacimiento a través de métodos informáticos con tecnología de última generación. El modelamiento puede ser realizado utilizando variables litológicas, de leyes y parámetros geotécnicos, lo importante es generar modelos representativos con un buen soporte geológico de las variables en evaluación.

  • Categorización de recursos minerales.
