ADMine System Suite

  • Los datos geológicos son la parte primordial para los modelos de recursos y las evaluaciones de recursos / reservas mineras. Es fundamental para las empresas mineras y de exploración proteger esta inversión con sistemas modernos para controlar el proceso y almacenar de forma segura los datos.

    ADMine system ofrece una amplia gama de módulos integrados para captura de datos, análisis, almacenamiento y generación de informes de datos geológicos, así como información geotécnica y geoquímica.

  • Módulo Drilling

    El Módulo Drilling es la herramienta única para todas las necesidades de gestión de taladros, incluida la planificación, el registro, la visualización y la gestión de todos sus datos relacionados con la profundidad final del pozo. Lo abarca todo, no es necesaria otra herramienta, aplicado a sondajes de exploración, producción, geología, geoquímica, geotecnia, geofísica, hidrogeología o cualquier otro campo relacionado.

    El Módulo Drilling le proporciona todas las herramientas necesarias para la recopilación de datos rápida y precisa, y con su capacidad avanzada para configurarlo para cumplir con todos sus requisitos de perforación específicos.

  • Módulo Exploration

    Almacene sus datos de muestra de campo en Módulo Exploration, un muestreo de campo independiente versátil en tiempo real que permite recolectar y almacenar datos de roca, suelo y otros materiales.

    Introduzca sus datos para obtener una base de datos completamente confiable y sin problemas.

  • Módulo Mine

    El Módulo Mine es un producto independiente que facilita el mapeo de características geocientíficas en todos los entornos mineros. Utilizando las prospecciones mineras existentes en operaciones a cielo abierto y subterráneas, los geólogos asignan las características de la mina de manera precisa y rápida directamente del sitio de trabajo.

    Estos datos asignados se usan directamente en los sistemas de planificación y modelado de su mina. Obteniendo una base de datos completamente confiable y sin problemas.

  • Módulo Laboratory

    El Módulo Laboratory tiene una integración directa con la base de datos geológicos para verificar de manera eficiente las muestras y enviar los resultados a los geólogos iniciadores; otros clientes de laboratorio también tienen una interfaz para solicitar el muestreo y recibir resultados.

    Los procedimientos establecidos entre el laboratorio y el cliente dictan automáticamente las reglas a seguir en las solicitudes de muestra.

  • Report

    El Módulo Report es un completo paquete de repositorio y de gestión de datos geológicos, geoquímicos, geotécnicos, geofísicos, de control de calidad y garantía de calidad (QA/QC), mapeo, topografía y otros datos de campo. Siendo completamente personalizable.

    Esto le permite personalizar el conjunto para adaptarlo específicamente a sus necesidades y procesos, al tiempo que abarca todas las funciones de administración de datos, informes y análisis que el ADMine tiene para ofrecer.
